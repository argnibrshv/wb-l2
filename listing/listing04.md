Что выведет программа? Объяснить вывод программы.

```go
package main

func main() {
	ch := make(chan int)
	go func() {
		for i := 0; i < 10; i++ {
			ch <- i
		}
	}()

	for n := range ch {
		println(n)
	}
}
```

Ответ:
```
Программа выведет числа от 0 до 9 включительно, и fatal error: 
all goroutines are asleep - deadlock!
Открытый пустой канал будет блокировать горутину, пока в него не будет переданы данные. 
Цикл for range при обходе канала, будет продолжаться пока канал не будет закрыт.
В данном коде после выполнения всех горутин, отправляющих число в канал ch, канал ch будет 
блокировать выполнение в горутине main, так как все данные из него были получены 
и данные в канал больше не передаются, но канал остается открытым и ожидает данные, что 
приводит к бесконечной блокировке в главной горутины.
```
